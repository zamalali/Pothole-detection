# Pothole-detection
Pothole detection using yolo v7 with pytorch.
Upload this ipynb file on colab and execute the cells one after another.
You can download the dataset from this link: https://www.kaggle.com/datasets/sovitrath/pothole-detection-yolov4-format
I have used Yolo v7 version on top of pytorch. You can view the output on my youtube channel : https://www.youtube.com/watch?v=DObED2beMIg
